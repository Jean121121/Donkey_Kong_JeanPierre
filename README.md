# Donkey Kong JeanPierre

## Descripción
Proyecto de videojuego desarrollado en Unity basado en Donkey Kong.

## Mejoras realizadas
- Nuevos modelos 3D (barriles, personajes, escenario)
- Implementación de scripts (BarrelController, BarrelSpawner)
- Mejora de físicas y colisiones
- Corrección de errores
- Mejora del gameplay

## Controles
- A / ← : mover izquierda
- D / → : mover derecha
- Espacio : saltar

## Cómo ejecutar
Abrir el proyecto desde Unity Hub.
